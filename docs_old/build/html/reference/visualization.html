circle_bundles.show_pca(data, *, colors=None, U=None, size=2.0, titles=None, max_points=2000, max_components=50, use_randomized=False, n_iter=4, jitter_eps=0.001, tol_flat_z=1e-06, random_state=0, set_cmap='viridis', set_cmap_range=(0.1, 0.9), show=True, return_figs=False)[source]

PCA visualization. :rtype: Tuple[‘go.Figure’, ‘plt.Figure’] | None

  • Fits PCA on the full dataset (up to max_components).

  • Plots first 3 PCs using Plotly (subsampled to max_points).

  • Plots cumulative explained variance using Matplotlib.

  • Prints CEV for k=1,2,3 (if available).

Optional: - If U is provided, renders one toggleable trace per set. - If colors is None and U is provided, colors are drawn from a Matplotlib colormap.

Parameters:

  • colors (Optional[Sequence])

Display patch thumbnails in a grid by (j,k) with graph edges drawn behind.

Assumptions

  • data is (N, d)

  • vis_func returns either a Matplotlib Figure OR an image array (H,W,3/4)
